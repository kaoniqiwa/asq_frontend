这些选项是什么意思

是 有时是 否 帮助
1.孩子有时候会发出喉音或“咯咯”声吗？
2.孩子会发出“哦” “咯” 和“啊”等咕咕声吗?
3.当您对孩子说话时,他/她能发出声音回应您吗?
4.当您对孩子说话时,他/她会对您微笑吗?
5.孩子会轻声咯咯笑吗？
6.当您离开宝宝的视线,然后又出现在他/她面前时,他/她会笑起来或显得高兴吗？
是 有时是 否 帮助
1.当孩子面朝上躺着时，他/她能挥动双腿和双臂并晃动和扭动身体吗？
2.当孩子面朝下趴着时，他/她能把头转到侧面吗？
3.当孩子面朝下趴着时，他/她能抬起头并持续几秒钟吗？
4.当孩子面朝上躺着时，他/她能踢双腿吗？
5.当孩子面朝上躺着时，他/她既能将头转向左侧又能将头转向右侧吗？
6.孩子趴着、抬着头时，他/她能把头向下放回到床上或地板上，而不是无力地跌落或垂落到床上或地板上吗？
是 有时是 否 帮助
1.当孩子醒着时，他/她的手经常是紧紧地握成拳头吗？如果以前是，但现在不再是了，也请勾选“是”。
2.当您用手指触摸孩子的手心时，他/她会抓住您的手指吗？
3.当您把玩具放在孩子手里时，他/她能握一会儿吗？
4.孩子的手能碰到自己的脸吗？
5.当孩子醒着时，他/她的双手是全部展开或部分展开的吗？通常孩子刚出生时是双手握拳的。
6.孩子会抓挠自己身上的衣服吗？
是 有时是 否 帮助
1.孩子能注视20～25厘米远处的物体吗？
2.当您在孩子周围走动时，他/她的眼睛能跟着您移动吗？
3.当您将玩具在孩子面前约25厘米远处缓慢地左右移动时，他/她的眼睛，或者是转动头使眼睛跟着玩具移动吗？
4.当您将玩具在孩子面前约25厘米远处缓慢地从他/她的头部到胸部移动时，他/她的眼睛能跟随玩具移动吗？
5.当孩子被抱着坐着时，他/她会注视在他/她面前桌子上或地板上的杯子或拨浪鼓大小的玩具吗？
6.当孩子面朝上躺着时，您将玩具在他/她的上方晃动，他/她能朝玩具挥舞手臂吗？
是 有时是 否 帮助
1.在没有喂奶时，孩子有时候也会吸吮嘴巴吗？
2.当孩子饿了、尿布湿了、累了或想让您抱的时候，他/她会哭吗？
3.孩子会对您微笑吗？
4.当您对孩子微笑时，他/她会用微笑来回应您吗？
5.孩子会看自己的手吗？
6.当孩子看见乳房或奶瓶时，他/她知道就要给他/她喂奶了吗？
