Q： 如果注册时宝宝信息填错了怎么办？

A： 在完成问卷前，宝宝信息都可以修改。如果提交了问卷，宝宝的年龄将不可修改，如果想要修改可以再添加一个新的宝宝信息。

Q： 家长的账户里可以同时注册几个宝宝呢？

A： 用户账号可以添加3个宝宝信息

Q： 家长答题的时候需要注意什么？

A： 1.答题过程请勿退出，之前填写的答案将都会清空
2.完成问卷后需要确认每一道题都完成问答，如果有答题遗漏系统会提醒用户完成

Q： 关于打印功能

A： 打印用户结果页和游戏活动页可以在宝宝库右侧进行选择。一般来说，打印页面会自动选择常规连接的打印机，若连接不正确，注意调整。另外如果想另存为pdf文档，也可以在这里选择后保存。

Q： 一个手机号有多个孩子怎么办（多胎）？

A： 如果家长有多个孩子并且希望只用一个手机来完成答题，有两种方法可以解决这个问题
1.如果是用电脑答题，可以在企业端的“添加宝宝”中“直接答题”选择“新用户”
2.如果是用手机答题，在“添加宝宝”中选择“ASQ”或“ASQ：SE”

在手机端页面中的右上角可以添加新的宝宝信息（一个手机号可以添加3个宝宝）
